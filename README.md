# ZeroClaw Configuration

Персонализированная конфигурация ZeroClaw для пользователя root.

## 📁 Структура
- `config/` — файлы конфигурации агента
- `skills/` — установленные навыки
- `tools/` — кастомные инструменты
- `scripts/` — скрипты восстановления
- `docs/` — документация
- `memory/` — ежедневные заметки

## 🚀 Быстрый старт
```bash
git clone git@github.com:Zhandos77/zeroClaw.git
cd zeroClaw
./scripts/restore.sh
```

## 🔧 Особенности
- Telegram бот с обработкой голосовых сообщений
- Парсинг PDF/DOCX документов
- Курсы валют для Казахстана и России (команды `/kz`, `/ru`, `/convert`)
- Персонализированные настройки и память
- Автоматическое обновление README и других файлов

## 📋 Содержимое

### Конфигурация
- `AGENTS.md` — правила поведения агента
- `SOUL.md` — идентичность агента  
- `USER.md` — пользовательские настройки
- `TOOLS.md` — заметки по инструментам
- `IDENTITY.md` — идентификация агента
- `MEMORY.md` — долгосрочная память

### Навыки
- `assistant/` — базовый ассистент для Telegram
- `currency/` — получение курсов валют

### Скрипты
- `restore.sh` — восстановление системы
- `pack_backup.sh` — создание архива бэкапа

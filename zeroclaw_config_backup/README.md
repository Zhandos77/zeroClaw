# ZeroClaw Configuration Repository

## 🦀 Моя персонализированная конфигурация ZeroClaw

### 📁 Структура репозитория

```
zeroclaw_config_backup/
├── config.toml              # Основная конфигурация ZeroClaw
├── README.md               # Этот файл
├── SETUP.md               # Инструкция по восстановлению
├── AGENTS.md              # Правила поведения агента
├── SOUL.md                # Идентичность агента
├── USER.md                # Пользовательские настройки
├── TOOLS.md               # Заметки по инструментам
├── IDENTITY.md            # Идентификация агента
├── MEMORY.md              # Долгосрочная память
├── skills/                # Установленные навыки
│   ├── assistant/         # Ассистент для Telegram
│   └── currency/          # Курсы валют
├── voice_processor.py     # Обработчик голосовых сообщений
├── telegram_voice_handler.py # Интеграция с Telegram
├── pdf_parser.py          # Парсер PDF документов
├── docx_processor.py      # Обработчик DOCX файлов
└── currency_helper.py     # Помощник по валютам
```

### 🔧 Установленные навыки

#### 1. Assistant (активен)
- **Описание:** Базовый ассистент для Telegram
- **Возможности:** Веб-поиск, выполнение команд, получение реальных данных
- **Правила:** Всегда прикладывает прямые ссылки, не выдумывает контент

#### 2. Currency (активен)
- **Описание:** Получение актуальных курсов валют
- **Источники:** НБ РК (Казахстан), ЦБ РФ, Open Exchange Rates
- **Команды:** `/kz`, `/ru`, `/convert <сумма> <из> <в>`

### 🎤 Обработка голосовых сообщений

**Установленные компоненты:**
- Vosk 0.3.50 (библиотека распознавания речи)
- Русская модель small-ru-0.22 (87MB)
- Telegram интеграция для автоматической расшифровки

**Поддерживаемые форматы:**
- Telegram голосовые (OGG Opus)
- WAV файлы (16kHz, mono)
- MP3 (с конвертацией)

### 🚀 Восстановление конфигурации

1. **Скопировать файлы:**
```bash
cp -r zeroclaw_config_backup/* /root/.zeroclaw/
```

2. **Установить зависимости:**
```bash
pip3 install PyPDF2 python-docx vosk
```

3. **Скачать модель Vosk:**
```bash
wget https://alphacephei.com/vosk/models/vosk-model-small-ru-0.22.zip
unzip vosk-model-small-ru-0.22.zip
```

4. **Настроить Telegram бота:**
- Получить токен у @BotFather
- Добавить в config.toml
- Добавить пользователя в allowed_users

# TTS Skill для ZeroClaw

Навык преобразования текста в речь (Text-to-Speech) с использованием Google TTS API.

## Быстрый старт

### Установка зависимостей:
```bash
pip install gtts pydub
```

### Использование в коде:
```python
from skills.tts.tts_skill import tts_convert, tts_send, tts_cleanup

# Конвертация текста в речь
audio_path = tts_convert("Привет, мир!", lang="ru")

if audio_path:
    # Отправка аудиофайла
    success, message = tts_send(audio_path)
    print(f"Результат: {success}, сообщение: {message}")
    
    # Очистка временного файла
    tts_cleanup(audio_path)
```

### Команды в Telegram:
- `/tts Привет, как дела?` - озвучить текст на русском
- `/tts Hello world lang=en` - озвучить на английском
- `/tts Сәлем дүние lang=kk` - озвучить на казахском

## Поддерживаемые языки

| Язык | Код | Пример |
|------|-----|--------|
| Русский | `ru` | Привет, мир! |
| Английский | `en` | Hello, world! |
| Казахский | `kk` | Сәлем, дүние! |
| Немецкий | `de` | Hallo, Welt! |
| Французский | `fr` | Bonjour, le monde! |
| Испанский | `es` | ¡Hola, mundo! |

Полный список языков: https://gtts.readthedocs.io/en/latest/module.html#languages-gtts-lang

## Параметры команды `/tts`

```
/tts <текст> [lang=<код_языка>] [speed=<скорость>] [tld=<домен>]
```

- **текст**: Текст для озвучивания (обязательно, до 1000 символов)
- **lang**: Код языка (по умолчанию 'ru')
- **speed**: Скорость речи (0.5-2.0, по умолчанию 1.0)
- **tld**: Домен Google ('com', 'ru', 'kz' и т.д.)

## Примеры

### Простые:
```
/tts Доброе утро!
/tts Сегодня хорошая погода
/tts Напоминаю о встрече в 15:00
```

### С параметрами:
```
/tts Hello world lang=en
/tts Guten Tag lang=de
/tts Buenos días lang=es speed=1.2
/tts Привет lang=ru tld=ru
```

### Длинные тексты:
```
/tts Это тест системы преобразования текста в речь. Система использует Google TTS API для генерации качественной речи на разных языках.
```

## Ограничения

1. **Длина текста**: Максимум 1000 символов (ограничение Google TTS)
2. **Интернет**: Требуется подключение к интернету
3. **Скорость**: Может быть задержка при генерации длинных текстов
4. **Качество**: Зависит от Google TTS сервиса

## Устранение неполадок

### Ошибка "Библиотека gTTS не установлена"
```bash
pip install gtts
```

### Ошибка "Нет подключения к интернету"
- Проверьте подключение к интернету
- Google TTS требует доступ к сервисам Google

### Ошибка "Язык не поддерживается"
- Проверьте код языка (должен быть из списка поддерживаемых)
- Попробуйте другой язык или домен

### Аудио не воспроизводится
- Проверьте поддержку MP3 формата
- Попробуйте другой плеер
- Убедитесь, что файл не поврежден

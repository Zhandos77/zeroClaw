# TTS Skill

Преобразование текста в речь через edge-tts и отправка аудио в Telegram.

## Команда

### `/tts <текст>`

**Алгоритм выполнения (строго следовать):**

1. Сгенерировать уникальное ASCII имя файла:
   ```bash
   OUTFILE="/tmp/tts_$(cat /proc/sys/kernel/random/uuid | tr -d -).mp3"
   ```

2. Синтезировать речь:
   ```bash
   edge-tts --voice ru-RU-SvetlanaNeural --text "ТЕКСТ" --write-media "$OUTFILE"
   ```

3. Убедиться что файл создан:
   ```bash
   ls -la "$OUTFILE"
   ```

4. Отправить файл — ответить ТОЛЬКО этой строкой:
   ```
   [AUDIO:/tmp/tts_XXXX.mp3]
   ```

**Важно:**
- Имя файла ВСЕГДА только латинские буквы, цифры и дефисы — НИКАКОЙ кириллицы
- Путь ВСЕГДА абсолютный `/tmp/tts_....mp3`
- После `[AUDIO:...]` никакого дополнительного текста
- Если edge-tts вернул ошибку — сообщить об ошибке текстом

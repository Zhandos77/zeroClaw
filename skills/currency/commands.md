# Currency Skill - Команды для ZeroClaw

## Основные команды

### `currency kz`
Получить курсы валют для Казахстана (НБ РК)

**Пример:**
```
currency kz
```

**Ответ:**
```
📊 Курсы валют (Казахстан) на 2026-03-11
Источник: https://ifin.kz/nbrk

USD (Доллар США): 1 USD = 489.39 KZT
EUR (Евро): 1 EUR = 567.99 KZT
RUB (Российский рубль): 1 RUB = 6.19 KZT
CNY (Китайский юань): 1 CNY = 71.28 KZT
GBP (Фунт стерлингов): 1 GBP = 657.98 KZT
```

### `currency ru`
Получить курсы валют для России (ЦБ РФ)

**Пример:**
```
currency ru
```

### `currency convert <сумма> <из> <в> [страна]`
Конвертировать сумму между валютами

**Параметры:**
- `<сумма>`: число для конвертации
- `<из>`: исходная валюта (например, KZT, USD, EUR)
- `<в>`: целевая валюта (например, USD, KZT, RUB)
- `[страна]`: опционально, KZ или RU (по умолчанию KZ)

**Примеры:**
```
currency convert 1000 KZT USD
currency convert 100 USD KZT
currency convert 1000 RUB USD ru
currency convert 500 EUR USD kz
```

**Ответ:**
```
💱 Конвертация:
1000 KZT = 2.04 USD
Курс: 1 USD = 489.39 KZT
Дата: 2026-03-11
Страна: KZ
```

## Сокращённые команды

### `/kz`
Алиас для `currency kz`

### `/ru`
Алиас для `currency ru`

### `/convert <сумма> <из> <в> [страна]`
Алиас для `currency convert`

## Поддерживаемые валюты

### Для Казахстана (KZ):
- **KZT** - Казахстанский тенге (базовая)
- **USD** - Доллар США
- **EUR** - Евро
- **RUB** - Российский рубль
- **CNY** - Китайский юань
- **GBP** - Фунт стерлингов
- **JPY** - Японская иена
- **TRY** - Турецкая лира

### Для России (RU):
- **RUB** - Российский рубль (базовая)
- **USD** - Доллар США
- **EUR** - Евро

## Особенности

1. **Кэширование:** Курсы кэшируются на 1 час для уменьшения количества запросов
2. **Автоопределение страны:** По умолчанию используется Казахстан (KZ)
3. **Обработка ошибок:** При недоступности источников используются кэшированные данные
4. **Форматирование:** Ответы форматируются для удобного чтения в чате

## Техническая информация

**Файлы скила:**
- `currency_simple.py` - основная реализация
- `__init__.py` - интерфейс для импорта
- `demo.py` - демонстрация работы
- `test_currency.py` - тесты

**Зависимости:**
- Только встроенные библиотеки Python (urllib, re, json, datetime)
- Не требует установки дополнительных пакетов

**Источники данных:**
- Казахстан: https://ifin.kz/nbrk
- Россия: https://yandex.ru/finance/currencies/USD_RUB
